  1. Definición

    Es la ciencia que estudia las propiedades, el comportamiento y las interacciones de la materia y la energía en el espacio y el tiempo. Analiza fenómenos naturales a través de principios y leyes fundamentales, utilizando el método científico.

    Estudia 4 elementos: materia, Energía, espacio y Tiempo.

  2. Medición

    Es el proceso de comparar una magnitud física con una unidad establecida de referencia para obtener un valor numérico que la describa. Es fundamental en la física, ya que permite cuantificar fenómenos y hacer predicciones.

  3. Unidades Fundamentales

    Son aquellas unidades de medida que no se definen en función de otras, sino que sirven como base para definir todas las demás. En el Sistema Internacional de Unidades (SI), las siete unidades fundamentales son:
    Metro (m) => Longitud
    Kilogramo (kg) => Masa
    Segundo (s) => Tiempo
    Amperio (A) => Corriente eléctrica
    Kelvin (K) => Temperatura
    Mol (mol) => Cantidad de sustancia
    Candela (cd) => Intensidad luminosa
